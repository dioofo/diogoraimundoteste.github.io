PROJECT OVERVIEW

Imminence Website Redesign

This concept project reimagines the digital presence of Imminence, a Swedish metalcore band known for blending heavy riffs with orchestral elements and emotional depth. The goal was to create a dark, immersive user experience that reflects the band's cinematic sound and visual identity.

This is a conceptual redesign of the official website for Imminence.The project was driven by a critical analysis of the band's existing site, which presented several UX and visual limitations.
